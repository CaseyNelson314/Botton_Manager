# Botton_Manager
## ボタンとカーソルの当たり判定取得クラス

![image](https://user-images.githubusercontent.com/91818705/158431943-f4ae09bc-6802-4d91-93c3-621afafd8613.png)

## Function
- `setPoint(x, y, _x, _y)` ボタンの座標設定rect関数などと同様に使用可
- `bottonMode()` 座標指定設定
- `hover()` カーソルが上にあるかどうか
- `click()` ボタン後された瞬間かどうか
- `Press()` ボタンが押されているかどうか
- `drawBotton()` 当たり判定のある範囲を表示
